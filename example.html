PicasaFetch example

Read the HTML source. The part you need to know is: 
picasawebGallery(picasa user, gallery name, element id);
where:
picasa user
is the name of the user the gallery belongs to
gallery name
is the internal gallery name. You can see it in the URL when you open the gallery in Picasaweb website. The URL will contain something like <user>/<gallery name>
element id
element where the table will be created. The element's content will be totally replaced, so you can leave a link for the gallery there, just in case the gallery loading fails for some reason.
This div will contain your gallery.
